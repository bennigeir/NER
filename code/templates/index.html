NER for Icelandic - API

A fine tuned BERT multilingual model for NER for Icelandic at your service.

Open endpoints require no Authentication.

NER : GET ?query=

Example: www.ice-bert-ner.com?query=Erna Sif er lektor við verkfræði- og tölvunarfræðideildir HR og forstöðumaður Svefnseturs sem nýlega var sett á fót með styrk frá Innviðasjóði.
